# Java Lab 实验仓库

这是一个用于长期、规范地管理 Java 小实验项目的仓库。

## 目录结构

- `projects/`: 所有的实验项目
- `shared/`: 可复用的工具类和算法库
- `templates/`: 项目创建模板
- `docs/`: 仓库文档、索引、笔记以及踩坑记录

## 管理规则

1. 所有实验项目必须放到 `projects/`
2. 可复用代码必须逐渐沉淀到 `shared/`
3. 每次创建新项目后，需在 `docs/index.md` 中维护索引
4. 项目必须包含 `README.md` 和 `notes.md`
